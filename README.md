## Projetos do Curso de Programação Web

Repositório contendo projetos e exercícios desenvolvidos durante o curso de *Programação Web*, com foco na aplicação prática dos conteúdos estudados.

### Tecnologias utilizadas
- HTML
- CSS
- JavaScript (introdução)
- Lógica de programação

### Objetivo do repositório
- Aplicar os conceitos aprendidos em aula
- Praticar a criação de páginas web
- Desenvolver raciocínio lógico para programação
- Consolidar a base para projetos futuros em Front-end

Os projetos representam minha evolução no aprendizado e minha dedicação à área de tecnologia.
